CELEBRITY IMAGES SETUP INSTRUCTIONS
====================================

To add celebrity images to this quiz app:

1. Download celebrity images from the internet (use free image sources like Unsplash, Pexels, or similar)
2. Save the images with the following filenames in this folder:

   Required Images:
   - tom_hanks.jpg
   - robert_downey_jr.jpg
   - beyonce.jpg
   - michael_jackson.jpg
   - oprah_winfrey.jpg
   - lebron_james.jpg

3. Image Requirements:
   - Format: JPG or PNG
   - Recommended size: 400x400 pixels or larger (square format works best)
   - File size: Keep under 1MB per image for better performance

4. After adding images, rebuild the app in Android Studio

Note: Make sure image filenames match exactly (case-sensitive) with the names used in CelebrityQuestionsActivity.java
